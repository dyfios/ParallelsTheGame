Introducing Parallels: The Ultimate Sandbox Game

Parallels is an in-development indie game project. The goal of this project is to enable the creation and exploration of collaborative high-fidelity virtual worlds that can be shared with friends (and strangers) across the Internet! Parallels blurs the line between a video game engine and a video game by giving players full control over every aspect of their world.

Key Features

    The following is a high-level list of current and planned features (it is by no means complete):
  • User-Defined Worlds, Entities and Physics
  • Virtually Endless* Procedurally-Generated Worlds
  • Language-Agnostic Scripting Infrastructure
  • Multiplayer System
  • Cross-Platform Support (Currently Windows x64 Desktop and SteamVR)
*Endless as far as data storage and computational resources will allow (in simpler terms: capable of generating worlds far larger than you'll likely ever be able to use).

Development Status

Despite the fact that Parallels is still in the pre-Beta (Alpha?) stage, I am aiming to make it available to anyone interested in using it as features are implemented. Please note that with this being very early software, there may be bugs.

How much does it cost to download? Nothing. It's FREE. In exchange, please provide feedback/ideas and let me know about any bugs that you encounter (I will be setting up a GitHub Issues board soon).
